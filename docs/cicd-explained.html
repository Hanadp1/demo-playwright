üîß Our Real Pipeline File (test.yml)

# This file tells GitHub Actions what to do
name: Playwright Tests CI

# WHEN should the robot wake up?
on:
  push:                      # When code is pushed
    branches: [ main ]
  pull_request:              # When someone wants to add code
    branches: [ main ]
  workflow_dispatch:         # Manual button to start!

# WHAT should the robot do?
jobs:
  test:
    runs-on: ubuntu-latest  # Use a Linux computer

    steps:
      - name: Checkout code
        uses: actions/checkout@v4         # Step 1: Download code

      - name: Set up JDK 11
        uses: actions/setup-java@v4       # Step 2: Install Java
        with:
          java-version: '11'

      - name: Install dependencies
        run: mvn clean install -DskipTests  # Step 3: Get libraries

      - name: Install Playwright browsers
        run: mvn exec:java ... install chromium  # Step 4: Install Chrome

      - name: Run tests
        run: mvn test -Dheadless=true      # Step 5: Test everything!

      - name: Upload test reports
        uses: actions/upload-artifact@v4   # Step 6: Save results
        with:
          name: test-reports
          path: target/surefire-reports/

üìö What Each Part Means:

  • üìã name: The name of our pipeline (like naming your pet!)
  • üîî on: When should the robot start working?
  • üíº jobs: The list of tasks to do
  • üñ•Ô∏è runs-on: What kind of computer to use
  • üë£ steps: Each action, one by one
